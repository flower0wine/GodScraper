# Telegram Scraper

A Python-based tool for scraping messages and media files from Telegram channels.

## What It Does

- Fetches messages from Telegram channels and groups
- Downloads media files (photos, documents) locally
- Stores data in SQLite database
- Exports data to CSV or JSON format
- Supports continuous scraping mode

## Quick Start

```bash
pip install -r requirements.txt
python main.py
```
